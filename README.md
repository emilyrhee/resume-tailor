# Resume Tailor

Resume Tailor is a full-stack web application designed to help users tailor their resumes to specific job descriptions or analyze candidates against job requirements. It uses a Retrieval-Augmented Generation (RAG) pipeline to fetch relevant context from a candidate's resume and generate targeted responses.

## Tech Stack

- **Frontend:** SvelteKit, Vite, Tailwind CSS, TypeScript
- **Backend:** Python, Flask
- **AI/RAG Pipeline:** LangChain, Cohere (Embeddings and Chat Models)
